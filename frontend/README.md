## Getting Started

##### Running locally in development mode

```bash
npm install
npm run dev

# ---- or

yarn
yarn dev
```

Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.

------
##### Building and deploying in production
```bash
npm install
npm run build
npm start

# ---- or

yarn
yarn build
yarn start
```

Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.
